# System Design Mastery

FAANG system design interview prep tracker with 30 problems, 20 building blocks, 15 architecture flows, and the RESHADED framework.

## Features

- 🗺️ **Roadmap** — 8-week curriculum
- 🧱 **Concepts** — 20 building blocks with use-when / don't-use
- 🏗️ **Problems** — 30 FAANG problems with step-by-step solutions
- 🎨 **Flows** — 15 visual architecture diagrams (including 5 AI system designs)
- 🎯 **Method** — RESHADED 8-step framework + grading rubric
- 📊 **Cheat Sheet** — estimation numbers, comparison tables
- 📚 **Resources** — curated courses, books, YouTube, mocks
- ⭐ Weak area tracking + notes per problem
- 💾 Progress saved in browser localStorage

## Getting Started

```bash
npm install
npm start
```

Opens at http://localhost:3000

## Build for Production

```bash
npm run build
```

Output in `build/` folder — deploy to any static host (Vercel, Netlify, GitHub Pages).

## Architecture Flows Covered

**Classic systems:** URL Shortener, Rate Limiter, Notifications, Twitter, Instagram, WhatsApp, YouTube, Uber, Dropbox, Google Docs

**AI systems:** ChatGPT/LLM, RAG, AI Image Generation, ML Recommendation, AI Agent with Tools

## Tech

- React 18
- No UI framework — pure inline styles
- localStorage for persistence
- Google Fonts (IBM Plex Mono, Outfit, DM Sans)
